  Major Projects
Backstepping Controller Design for Variable Pitch Quadrotor
Course project under, Prof. Mangal Kothari and Prof. S R Sahoo, IIT Kanpur   (May-Jul 2016)
  • Modelled the quadrotor dynamics using simple blade element momentum theory and separated the dynamics in two subsystems i.e. rotational and translational dynamics
  • Used a cascade control structure i.e. inner loop for rotational dynamics and outer loop for translational dynamics in which control commands are generated by using backstepping control approach
  • Added an extra loop to the control structure to allocate the control commands(blade pitches) so that the nonlinear correlaton between pitch of the blades and yaw moment is taken care of
  • Demonstrated simulations for stabilization and trajectory tracking problem

Stereo Camera Calibration and Disparity Map Generation in Real Time
Course project under, Prof. Abhishek and Prof. Mangal Kothari , IIT Kanpur   (Jan-Apr 2016)
  • Developed a stereo camera system using two single Logitech webcams
  • Calibrated the stereo camera system and displayed disparity and depth map in real time using ROS image pipeline

  Selected Other Projects
Design and Fabrication of Coaxial Free-wing Helicopter
Course project under, Prof. A K Ghosh, IIT Kanpur   (Jan-Apr 2015)
  • Built an unconventional VTOL vehicle which consists of coaxial rotor and a wing allowing extensive maneuverability and increased thrust. It Was an innovative project taken up for the first time in the course
  • Demonstrated flight of the vehicle in hover mode
Model of Railway Switch
Course project under, Prof. Kantesh Balani, IIT Kanpur   (Jan-Apr 2014)
  • Manufactured a model of railway switch along with a railway engine from steel and iron material using several manufacturing processes such as casting, welding, brazing, metal forming etc.
  • With this model, we could demonstrate how the actual railway switch functions
  • Received Best Sectional Project Award for this design and fabrication project which was submitted as partial fulfillment for an introductory course on Manufacturing Processes
Coupled Trim Analysis of a Conventional and Tandem Helicopter
Project under, Prof. Abhishek, IIT Kanpur   (Jun-Nov 2014)
  • Developed a MATLAB code to perform coupled trim analysis of a helicopter based on Jacobian method
  • Used Newmark's method to numerically solve blade flapping equation
  • Obtained results for uniform inflow model and drees' model for inflow and compared these results with the available real flight data
  • Extendend the code to get trim controls for a tandem helicopter, in which only collective and lateral cyclic pitch of two rotors were trimmed, and longitudinal cyclic control input was varied as apriori known variation with the forward speed.
  • Used Uniform inflow model for the calculation of rotor forces
  • Compared the simualtion results for Chinook CH-47B with the existing reference data from the literature.
